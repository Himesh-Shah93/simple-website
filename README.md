# Simple Website

A clean and modern simple website template built with HTML, CSS, and JavaScript.

## Features

- **Responsive Design**: Works perfectly on desktop, tablet, and mobile devices
- **Modern Styling**: Clean and minimalist design approach
- **Smooth Navigation**: Smooth scrolling between sections
- **Contact Form**: Functional contact form with validation
- **Animations**: Subtle animations and transitions
- **Easy to Customize**: Well-organized code that's easy to modify

## File Structure

```
simple-website/
├── index.html      # Main HTML file
├── styles.css      # CSS styling
├── script.js       # JavaScript functionality
└── README.md       # This file
```

## Sections

1. **Navigation Bar** - Sticky navigation with links to all sections
2. **Hero Section** - Eye-catching welcome section
3. **About Section** - Information about the website with feature cards
4. **Services Section** - Display of offered services
5. **Contact Section** - Contact form for inquiries
6. **Footer** - Footer with copyright information

## Usage

1. Download or clone the repository
2. Open `index.html` in your web browser
3. Customize the content, colors, and text to match your needs

## Customization

### Colors
Edit the CSS variables in `styles.css`:
```css
:root {
    --primary-color: #4a90e2;      /* Main color */
    --secondary-color: #f5f5f5;    /* Background color */
    --text-color: #333;             /* Text color */
}
```

### Content
Edit the HTML in `index.html` to change:
- Website title and branding
- Navigation menu items
- Section content and text
- Contact form fields
